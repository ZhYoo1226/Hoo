```python
class UpdateTableHeadState(BaseState):
    """
    状态的功能描述注释
    更新表格头状态
    """
    stateName = "UpdateTableHeadState"

    def __init__(self,
                 #每一个参数必须有类型，参数注释说明
                 headers: list,  # 推测数据表头字段提示词返回的params属性
                 uuid: str,  # 文件uuid
                 **kwargs):
        super().__init__()
        self.headers = headers
        self.file_uuid = uuid
```

## Action描述结构

```json
{
  "action_id": "string (uuid, 可选, 用于追踪)",
  "action": "string (范畴:具体动作, 见动作注册表)",
  "url": "https://etax.chinatax.gov.cn" //这些参数直接是目标tools的参数
  "thought": "string (LLM的思考摘要, 便于调试与日志)",
  "confidence": 0.0-1.0,         // 可选, LLM对该动作的信心
  "expected_outcome": "string"   // 可选, 预期观察, 用于后续反思
}
```

所有的输出必须遵循如下规则：

```
# 真实数据

【用户提示词相关内容输入】

**输出:**
## 输出标签
```

- 输出标签，必须是##级别，这样方便内部进行解析。

举例：

````markdown
# 真实数据

**对话历史:**
```
{get_chat_history_text}
```

**输出:**
## 用户意图分发

  "thought": "用户情绪表达，需要继续聆听用户反馈。"
````
