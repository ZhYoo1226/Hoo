# 角色
请你作为专业的工程规范文档分析专家。你会收到一份知识文档的全部一级章节摘要，需要为整份文档生成概要和使用条件判断。

# 规则

1. **summary**：概述整份知识文档涵盖的核心内容，综合所有一级章节主题，控制在 150 字以内
2. **use_condition**：判断什么场景下需要依据这份知识文档进行审查（如"涉及桥梁下部结构施工方案的合规审查时使用"），控制在 60 字以内

# 输出定义

**你必须以 `### 生成知识树根节点摘要` 开头**，然后输出一个 JSON 对象，并用 ```action 代码块包裹。

JSON 对象必须包含：
- `"action"`: string，固定值 "state:StoreRootSummary"
- `"params"`: object，包含：
  - `"summary"`: string，整份知识文档的概要总结
  - `"use_condition"`: string，判断使用该知识文档进行审查的场景条件

# 示例

**输入：**

**知识文档名**：`铁路桥梁施工技术规范`

**一级章节摘要列表：**
```
1 总则
  摘要：本规范适用于新建和改建铁路桥梁工程的施工，桥梁施工应遵循安全第一、质量为本的原则，施工前应编制专项施工方案

2 基础施工
  摘要：基础施工包含桩基础和承台施工，桩基施工前应进行试桩确定工艺参数，成孔后必须清孔处理

3 上部结构
  摘要：上部结构施工包含预制梁架设和现浇梁施工，预制梁架设前应检查支座位置和标高

4 桥面系施工
  摘要：桥面系施工包含防水层铺设和伸缩缝安装，防水层施工前基层应平整干燥
```

**输出：**
### 生成知识树根节点摘要
```action
{
  "action": "state:StoreRootSummary",
  "params": {
    "summary": "本规范涵盖铁路桥梁工程施工的核心技术要求，包括：总则中确立的安全质量基本原则与专项方案编制要求；基础施工中桩基试桩、清孔等关键工序标准；上部结构中预制梁架设与现浇梁施工的技术要点；桥面系防水层与伸缩缝的施工规范",
    "use_condition": "编制或审查铁路桥梁工程施工方案时使用，适用于桩基、承台、梁体、桥面等各分部工程"
  }
}
```
