这里为你详细整理了 `lunar-python` 库的安装与使用方法。

### 📦 安装与环境要求

`lunar-python` 是一个纯净的Python库，配置起来非常简单。在终端中运行以下命令即可完成一键安装：

```bash
pip install lunar_python
```

> **环境要求**：Python **3.6及以上版本**（请注意，`lunar-python` `v1.2.23`版本之后已不再支持Python 2）。

### 🚀 快速上手：日期转换

`lunar-python` 主要提供了 `Solar` (公历) 和 `Lunar` (农历) 两个核心类，用于在两种历法间进行转换。

#### 示例一：公历转农历
你可以先创建一个公历日期对象，然后通过 `getLunar()` 方法获取对应的农历日期。

```python
from lunar_python import Solar

# 创建一个公历日期（2023年10月1日）
solar = Solar(2023, 10, 1)
print("公历日期:", solar.toDateString())  # 输出：公历日期: 2023-10-01

# 转换为农历日期
lunar = solar.getLunar()
print("农历日期:", lunar.toFullString())  # 输出：农历日期: 二零二三年八月十七 癸卯(兔)年 ...
```
> **关键方法**：`Solar` 类用于表示公历日期，`toDateString()` 格式化输出公历日期，`getLunar()` 转换为农历。

#### 示例二：农历转公历
你也可以将农历日期反向转换为公历。

```python
from lunar_python import Lunar

# 创建一个农历日期（2023年农历正月初一）
lunar_date = Lunar.fromYmd(2023, 1, 1)

# 转换为公历日期
solar_date = lunar_date.getSolar()
print("公历日期:", solar_date.toFullString())  # 输出：公历日期: 2023-01-22 00:00:00 星期日 水瓶座
```
> **关键方法**：`Lunar.fromYmd()` 通过指定年月日创建一个农历日期对象，其 `getSolar()` 方法用于转换为公历。

#### 示例三：获取当前农历日期
如果想获取系统当天的农历日期，可以结合 `datetime` 模块使用。

```python
from lunar_python import Solar
from datetime import datetime

# 获取系统当天的公历日期对象
today_solar = Solar.fromDate(datetime.today())
# 转换为农历并显示详细信息
today_lunar = today_solar.getLunar()
print(today_lunar.toFullString())
```

> `Solar.fromDate()` 方法可以直接接受Python的 `datetime` 对象来创建 `Solar` 实例，非常方便。

### 📑 核心功能介绍

除了基础转换，`lunar-python` 还内置了非常丰富的传统文化信息查询功能。

| 类别 | 功能与举例 | 代码调用示例（基于Lunar对象） |
| :--- | :--- | :--- |
| **干 支** | **年、月、日、时的天干地支组合** | `lunar.getYearGanZhi()` |
| **生 肖** | **年份对应的生肖** | `lunar.getYearShengXiao()` |
| **纳 音** | **年、月、日的纳音五行** | `lunar.getYearNaYin()` |
| **节 气** | **判断当前日期的节气** | `lunar.getJieQi()` |
| **传 统 节 日** | **获取传统节日名称** | `lunar.getFestivals()` |
| **宜 与 忌** | **彭祖百忌的内容** | `lunar.getPengZuHundredAvoid()` |
| **吉 神 方 位** | **喜神、福神、财神等方位** | `lunar.getHappyGodPosition()` |
| **冲 煞** | **冲什么生肖，煞哪一方** | `lunar.getClash()` |
| **星 宿** | **二十八星宿的名称和吉凶** | `lunar.getConstellation()` |
| **星 座** | **公历日期对应的星座** | `solar.getXingZuo()` |

> **注意**：`solar.getLunar()` 返回 `Lunar` 对象后，即可调用上述方法获取丰富的文化信息。

### 💡 高级功能

对于更专业的应用场景，`lunar-python` 也提供了相应的工具支持。

-   **八字分析 (`eightchar`)**：这个子模块提供了专业的八字计算功能，可以用于分析大运流年、流月运势等，适合命理研究相关的应用开发。
-   **实用工具 (`util`)**：`util` 目录下存放着一些实用的辅助工具类，例如 `HolidayUtil` 可以用来查询和管理法定节假日与调休安排。
