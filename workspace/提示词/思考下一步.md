## ⚠️ 关键反思规则
1. **错误处理**：如果收到的「观察结果」中包含“Error”字样，你**必须**在下一轮的「思考」中分析错误原因，并主动调整参数或更换其他工具，**不得直接放弃**。
2. **状态感知**：在执行下一步之前，务必先检查上一轮「观察结果」的内容。
3. **幂等性**：每一次工具调用都应使用唯一的 `id` 值，以便准确追踪对应的执行结果。

```json

    "path": "/data/"
  }
}
```
