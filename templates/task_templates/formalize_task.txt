请将最终答案以标题（按顺序）开头，标题分别为"### OBJECTS"（不带方括号）、"### INITIAL"和"### GOAL"，并在每个标题下的``` ```注释块中，按照PDDL格式分别包含相应内容，示例如下：

### OBJECTS
```
object1 - type_1 ; object1的注释
object2 - type_2 ; object2的注释
object3 - type_1 ; object3的注释
```

每个对象必须单独声明其类型，不得进行分组——即使多个对象属于同一类型。

### INITIAL
```
(<谓词名> <对象1> <对象2>) ; 初始状态谓词的注释
(<谓词名> <对象3> <对象4>) ; 初始状态谓词的注释
(<谓词名> <对象5>) ; 初始状态谓词的注释
(= (<函数名> <对象6>) <值>) ; 数值赋值的注释
```

### GOAL
```
(<谓词名> <对象>) ; 目标状态谓词的注释
(<操作符> (<函数名> <对象>) <值>) ; 数值赋值的注释
```
